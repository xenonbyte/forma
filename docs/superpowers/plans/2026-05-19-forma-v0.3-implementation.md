# Forma v0.3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the approved Forma v0.3 batch: product language config, unified requirement saving, structured multilingual copy, BDD rules, patch/rebuild design flow, Web localization, updated Agent commands, and documentation.

**Architecture:** Implement contracts in `packages/core` first, then expose them through MCP and HTTP adapters, then update Web and Agent consumers. Keep AI semantic work in Agent templates; backend services validate structure, persist state, and enforce deterministic lifecycle gates. Use narrow services for copy and rules so `RequirementService` remains a state-machine coordinator rather than a storage grab bag.

**Tech Stack:** Node 22+, pnpm 10.33, TypeScript 5.9, Zod, js-yaml, Vitest 4.0.7, Fastify 5, React 19, Vite 7, Tailwind CSS v4, Pencil CLI integration.

export function assertProductConfig(

  productId: string,
  fields: Array<"platform" | "style" | "languages" | "components_initialized">

}

- `empty + ui_affected=true` sets all pages pending and writes translations.
- `submitted/active + ui_affected=true` preserves unchanged pages, sets `new` pending, sets `patch/rebuild` expired, preserves existing design ids, and replaces navigation.
- `ui_affected=false` updates document and rules without changing pages or baseline.
- Calling `saveRequirement` twice for the same requirement replaces that requirement's rules instead of duplicating them.
- `replaces_rule_id` and `remove_rule_ids` delete rules from other requirements.

- `ui_affected=false` rejects archived requirements with `REQUIREMENT_STATUS_INVALID`.
- UI save rollback restores all 5 transaction files when translations, document, or rules writes fail.
- Logic-only rollback restores all 3 transaction files when document or rules writes fail and does not snapshot or rewrite baseline/translations.
